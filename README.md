# Love You Like XO
A terminal tick-tack-toe game for the hopelessly erudite.

## To Run
- You'll need Python 3+
- Fork the repo into a fresh directory
- run `$ python3 play.py`

### To Play
- Respond to the prompts.
- All you can do right now is fill up the board.
